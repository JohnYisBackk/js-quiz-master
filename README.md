# Quiz Master PRO 🎯

Quiz Master PRO is an interactive quiz application built with **HTML**, **CSS**, and **Vanilla JavaScript**.

Users can start a quiz, answer multiple-choice questions, receive instant feedback, track their progress, and see their final score at the end.

## ⚙️ How It Works

The quiz stores questions in an array of objects. Each object contains a question, a list of possible answers, and the correct answer.

JavaScript tracks the current question index, user score, and whether an answer has already been selected. Based on the user’s answer, the app updates the UI, shows feedback, disables answer buttons, and moves to the next question.
